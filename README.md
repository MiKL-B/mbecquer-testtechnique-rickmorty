# rickmorty-app

## Clone the project

```
git clone https://github.com/mbecquer/mbecquer-testtechnique-rickmorty.git
```

### Go to the app

```
cd rickmorty-app
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
